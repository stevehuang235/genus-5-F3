This folder contains the table of $$\mathbf{F}_3$$-isomorphism classes of hyperelliptic curves of genus 5. 

- ```data_hyperelliptic.txt``` contains one representative for each isomorphism class. Each line is of the form ```[ point count, group label, function field polynomial representation]```. This table was generated by the Magma script ```isomcheck_datagen.m``` using the data in ```genus5_Howe``` computed and generously provided to us by Everett Howe. We verified that all the curves in ```genus5_Howe``` are pairwise non-isomorphic as an extra consistency check.
